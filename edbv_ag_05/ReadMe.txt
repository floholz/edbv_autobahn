1. 'MotorwayCarDetection.mlapp' in Matlab öffnen und ausführen.
2. Den Dateinamen (inklusive Dateiendung) des Videos im Textfeld des GUI's eingeben.
3. Durch Betätigen des Buttons "Measure Speed" wird das Progrmm ausgeführt.
   Durch Betätigen des Buttons "Play Original" wird das unbearbeitete Video abgespielt.

Der Code kann folgendermasen reviewed werden:
1. Den Befel "appdesigner('MotorwayCarDetection.mlapp')" im Command Window eingeben.
   Dadurch öffnet sich der "App Designer".
2. Auf den Tab "Code View" wechseln (Button im rechten oberen Bereich   ODER   Shift + F7).

   ACHTUNG: Das Video muss im gleichen Ordner wie MotorwayCarDetection.mlapp sein 
            UND 
            Es muss die Computer Vision System Toolbox installiert sein.
